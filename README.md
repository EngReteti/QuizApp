# QuizApp

A professional, console-based Quiz Application built with Java.

## Features
- Dynamic Question Loading
- Score Tracking
- Category Selection (Planned)

## Tech Stack
- **Language:** Java
- **Environment:** Termux (Android)
- **Version Control:** Git

## How to Run
```bash
javac src/main/java/com/quizapp/Main.java
java -cp src/main/java com.quizapp.Main
```
